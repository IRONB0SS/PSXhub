# PSXhub

This app is enhanced version of PSX Download Helper with more features and fixed bugs.

features :

• No more crash while transferring PKGs

• Transfer Retail PKGs to Your PS4/PS5 with two methods (Local Database & AutoFinder)

• Check downloaded PKGs health

• Online DataBase (Special Thanks to [0x199]([url](https://github.com/0x199)) helps and guidance)

• Transfer rate monitor (thanks to @zhongyang219 and his wonderful work [TrafficMonitor]([url](https://github.com/zhongyang219/TrafficMonitor)))
